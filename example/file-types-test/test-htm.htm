Test Page Alternative - .htm Extension

Purpose: This file validates that the Workspace Wiki extension treats both .html and .htm file extensions equivalently when HTML support is enabled.

Extension Equivalency Testing

This document serves to ensure that:

  • Both .html and .htm files are discovered together
  • The shorter .htm extension (legacy Windows format) works correctly
  • File ordering treats both extensions the same way
  • Opening and preview behavior is consistent across extensions

When set to false (default), neither should appear, keeping the tree focused on Markdown and plain text documentation.

Expected Behavior: This .htm file should be treated identically to .html files in all aspects of discovery, display, and interaction within the Workspace Wiki extension.
